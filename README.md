# 🎓 LearnMate – Agentic AI for Personalized Course Pathways

## 📌 Overview

**LearnMate** is an Agentic AI-powered learning assistant that helps students discover the most suitable learning pathway based on their interests, current skills, and career goals. Instead of browsing through thousands of online courses, students receive a personalized roadmap that evolves as they progress.

This project is developed as part of **Problem Statement No. 12 – Agentic AI for Personalized Course Pathways** and leverages **IBM Cloud Lite** and **IBM Granite** for AI-powered guidance.

---

## 🚀 Features

* 👤 Student Profile Creation
* 🧠 AI-Based Skill Assessment
* 🛣️ Personalized Learning Roadmap
* 📚 Course Recommendations
* 📊 Progress Tracking Dashboard
* 💬 AI Learning Coach (Chat Interface)
* ⚙️ User Settings
* 📱 Responsive Streamlit Interface

---

## 🎯 Problem Statement

Students often struggle to choose the right learning path because of the vast number of online courses and the lack of personalized guidance. LearnMate addresses this challenge by acting as an AI mentor that understands each student's interests and skill level, then generates a customized learning roadmap that adapts over time.

---

## 💡 Solution

LearnMate interacts with students to:

* Understand their interests and career goals.
* Assess their current knowledge and skills.
* Generate a personalized learning roadmap.
* Recommend relevant learning resources.
* Track progress and update recommendations dynamically.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/LearnMate.git
```

### 2. Move into the project directory

```bash
cd LearnMate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```

---

## 📖 Application Workflow

1. Student creates a profile.
2. AI analyzes interests and career goals.
3. Student completes a skill assessment.
4. LearnMate generates a personalized roadmap.
5. Recommended courses are displayed.
6. Student tracks progress through the dashboard.
7. AI continuously updates recommendations based on learning progress.
